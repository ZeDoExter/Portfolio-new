# Operation: Decrypt

This toolkit was born out of the necessity to solve complex cryptographic challenges efficiently during timed CTF competitions.

## Capabilities

- **Hidden Message Extraction**: Tools to identify and decode hidden Unicode patterns and non-standard character encodings.
- **Pattern Analysis**: Automated frequency analysis to break substitution ciphers.
- **Rapid Scripting**: A library of reusable Python snippets for common data manipulation tasks.

## Notable Solves

Used successfully to crack the "Invisible Ink" challenge, revealing a flag hidden entirely within zero-width characters in a seemingly normal text file.
