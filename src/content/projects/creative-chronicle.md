# Extra! Extra!

**The Creative Chronicle** is more than just a portfolio; it is a statement on the state of web design. Moving away from generic bootstrap layouts, this project embraces the "Newspaper" aesthetic.

## Design Philosophy

- **Retro-Modern Fusion**: Combining classic serif typography (Playfair Display, Old Standard TT) with smooth, modern GSAP animations.
- **Interactive Elements**: Draggable papers, floating text, and reactive layouts that respond to user interaction.
- **Dual Layout System**: Features two distinct layout modes (Retro Grid & Complex Magazine) that are deterministically selected based on project content.

## Tech Stack

Built with **SvelteKit** for the framework, styled with **CSS Variables** for easy theming (supporting both "Paper Beige" and "High Contrast White" modes), and animated with **GSAP ScrollTrigger**.
